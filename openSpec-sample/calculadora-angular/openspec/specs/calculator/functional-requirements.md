# Calculator App - Functional Requirements

## Goal
Build a web-based calculator with basic arithmetic operations using Angular.

## Core Features
- [ ] Display numeric input and results
- [ ] Basic operations: +, -, ×, ÷
- [ ] Clear (C) and backspace (⌫) functions
- [ ] Decimal point support
- [ ] Error handling (e.g., division by zero)

## Non-Functional
- [ ] Responsive design (mobile + desktop)
- [ ] Accessible (WCAG 2.1 AA)
- [ ] No external calculation libraries (pure TypeScript)

## Acceptance Criteria
- User can perform chained operations: 2 + 3 × 4 = 20
- Display updates in real-time as user types
- Clear button resets state completely
